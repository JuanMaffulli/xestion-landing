Capturas del sistema

Explora las pantallas principales del sistema para pasteleria.

Nueva Venta / Pedido

Formulario para crear ventas y pedidos de productos. Seleccion de cliente, productos del catalogo, calculo automatico de totales y gestion de estados de pago y entrega.

Historial de Ventas

Lista completa de todas las ventas realizadas con filtros por fecha, cliente, estado y importe. Exportacion a Excel y busqueda avanzada incluida.

Gestion de Clientes

ABM completo de clientes con datos de contacto, telefono, direccion y notas relevantes. Busqueda rapida por nombre y filtros.

Receta de Producto (Insumos)

Asignacion de insumos a cada producto con cantidades precisas. El sistema calcula automaticamente el costo de produccion segun precios actuales de materias primas.

Catalogo de Productos

Lista completa de productos con precios de venta, costos calculados segun receta, categorias y margenes de ganancia. Filtros por categoria y busqueda avanzada.

Lista de Insumos

Catalogo de materias primas e insumos con precios actualizables, unidades de medida, proveedores y stock disponible. Base fundamental para el calculo de costos.

Control de Caja

Registro de ingresos, egresos, cobros y pagos diarios. Control de efectivo, transferencias y metodos de pago con totales por periodo.

Historial de Cotizaciones

Presupuestos enviados a clientes con posibilidad de conversion a venta. Seguimiento de cotizaciones cerradas, pendientes y rechazadas.

Historial de Compras de Insumos

Registro de compras de materias primas a proveedores. Control de costos, fechas, cantidades adquiridas y actualizacion de precios de insumos.

Gestion de Proveedores

ABM de proveedores de materias primas con datos de contacto, rubros y condiciones de pago. Organizacion de compras por proveedor.

Cuenta Corriente de Proveedores

Seguimiento de deudas y pagos a proveedores con historial de movimientos, saldos pendientes y vencimientos de facturas.

Historial de Gastos

Registro de gastos operativos con categorias (alquiler, servicios, impuestos, etc.). Control del flujo de egresos y analisis de donde se va el dinero.

Lista de Tareas / Produccion

Control de produccion diaria con tareas pendientes, en proceso y completadas. Seguimiento de que se necesita elaborar para cumplir pedidos.

Reportes Personalizados

Analisis de rentabilidad por producto, ventas mensuales y anuales, productos mas vendidos, margenes de ganancia y estadisticas clave. Reportes totalmente personalizables.
